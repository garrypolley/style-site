Typography

This page contains examples of the various typographical styles defined with this UI guide.

Examples

Below you will see examples of various headings and the CSS needed to achieve each style. The size of the headings changes once you drop below the medium viewport width. This is on purpose to take up less space on smaller devices and make body text easier to read and standout more.
